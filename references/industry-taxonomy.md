# 国民经济行业分类参考（精简版）

> 基于 GB/T 4754-2017《国民经济行业分类》，提取门类→大类两级结构。
> 用途：将用户的模糊调研需求锚定到标准行业边界。

## 使用方法

1. 接收用户输入的行业关键词（如"低空经济"、"新能源"）
2. 在下表中匹配最相关的门类和大类
3. 一个用户需求可能跨多个门类（如"低空经济"涉及 C-制造业 + G-交通运输 + I-信息技术）
4. 输出 `industry_anchor` 时需列出所有相关的行业代码
